
# Wright Brothers Roofing Website

This repository contains the source code for the **Wright Brothers Roofing** website. The project is built with modern web technologies to deliver a fast, responsive, and visually engaging experience for visitors.

The site showcases roofing services, company information, and provides an easy way for potential customers to request estimates or contact the business.

# Getting Started (Local Development)

You can run this project locally using your preferred IDE.

### Requirements

Make sure you have the following installed:

* Node.js
* npm

Recommended installation method:

[https://github.com/nvm-sh/nvm#installing-and-updating]

---

### Installation Steps

Clone the repository and start the development server.

```bash
# Clone the repository
git clone https://github.com/Lishguy/wright-brothers-digital-ascent

# Navigate to the project directory
cd wright-brothers-digital-ascent

# Install dependencies
npm install

# Start the development server
npm run dev
```

Once the server starts, the project will be available locally with automatic reload when files are updated.
